# pong

A simple game of pong implemented using [PyGame](https://www.pygame.org/contribute.html).

## Installation:
``` 
git clone https://github.com/ariane-lozachmeur/pong.git
cd pong
pip3 install requirements.txt
```

## Launch a game
``` 
python3 main.py
```
